Portal interface header#

The header's elements and buttons

The header contains the elements that are common for all sections of the Portal. They are listed below from the header's left side to the right side:

  1. Menu button toggles navigation panel
  2. Search field is an input field for search queries
  3. List view button switches section content to a list view
  4. Grid view button switches section content to a grid view

  5. Map view button switches section content to a map view

    Note: This button is available only in the Projects section

  6. Profile settings button brings up the profile settings menu
