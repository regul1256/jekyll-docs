User roles and permissions in Portal#

This article is about project permissions tab

All the roles are managed by the administrator of the Portal. Any Portal project can have users with the following roles:

  • admin
  • manager
  • operator
  • user
  • viewer

Note

Contact your Portal administrator to get information about your user role.

Users with admin and manager roles have access to IDE functionality. Users with viewer role can also be able to work in IDE. Portal administrator can grant permission to such users.

Important

It is important to notice that mentioned above roles can only see a predefined set of pages and documents. The administrator of the Portal configures which pages are going to be available to one or another user.
