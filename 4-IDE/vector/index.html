Vector tags#

Vectors are virtual tags made for changing values of particular elements of array tags.

The Vector tag editor icon is in the Header Toolbar of the View tags menu. You can see the editor in the figure below.

You can set standard parameters: Name, Custom name, and Description in the respective fields.

There are three options for the Type parameter: Simple, Regulator link and Time channel.

The Source tag parameter ties the Vector tag with a source tag. The source tag should be of array data type.

The Index parameter defines the number of the element in the source tag that is going to be changed. The count starts with 0. So, the first element has index 0, the second – 1, the third – 2, and so on.

The Writable flag is inactive by default, but it should be active so users would be able to change values.
