Generic tag editor#

IDE supports editing several tags of different types at the same time. To open the Mass tag editor, choose several devices and tags in the VIEW TAGS menu and click MASS EDIT TAGS.

The Generic tag editor is an interactive table that allows you to edit tag properties. Tags of different types have different properties, and so you can edit only overlapping properties of a selection of tags.
