IDE#

An overview of ControlMachines™ HMI project development component.

IDE consists of:

  1. Page editor
  2. Device editors
  3. Tag editors
  4. Alarm editor

ControlMachines™ HMI Portal supports project editing. This functional block is called IDE. Having permissions to work with IDE, you can do the following:

  1. Create a page
  2. Add elements to the page
  3. Edit parameters and settings of the element
  4. Edit relations between elements
  5. Create, edit, and delete devices
  6. Create, edit, and delete tags
  7. Create, edit, and delete alarms
