Importing tags#

You can import tags from a .csv file. There is the import tags button in the header of the View tags menu. This button opens the import menu.

The Import tags menu contains drag and drop area, where you can put the importing file. And conflict strategy options:

  1. Update all items from the file
    Uploads all the tags from the .csv file. In the case of similarities, the same objects will be replaced by ones from the importing file.
  2. Update all items from the file and remove the rest
    Allows removing tags from the recipient folder while the others are updated from the importing file. Identical tags are to be replaced by ones from the file.
  3. Add new items from the file
    Choosing this option, you are going to add only those tags that were not present in the recipient folder.
  4. Do nothing in case of conflicts
    Displays an error message that contains the name of the first conflicting tag.

Note

When choosing tags for removing, hold the ctrl button to pick up several objects.
