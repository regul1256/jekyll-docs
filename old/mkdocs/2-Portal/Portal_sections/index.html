Portal sections#

A brief overview of Portal sections

Portal has two sections:

  • Projects is a section for monitoring available facilities. This section contains available projects. You can navigate to a project's HMI or open project profile menu.

  • Comparisons is a section that provides a user with the instrument for monitoring tags from different projects.
